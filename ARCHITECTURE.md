================================================================================
                    MIGRACION A AZURE CLOUD
================================================================================

  LOCAL (Azurite)                    CLOUD (Azure Blob Storage)
  =================                   ========================
  
  AZURITE_ACCOUNT_NAME=              AZURITE_ACCOUNT_NAME=
    devstoreaccount                  mystorageaccount
  
  AZURITE_ACCOUNT_KEY=             AZURITE_ACCOUNT_KEY=
    (default Azurite key)           <azure_storage_key>
  
  AZURITE_BLOB_ENDPOINT=           AZURITE_BLOB_ENDPOINT=
    http://azurite:10000/...         https://mystorageaccount
                                     .blob.core.windows.net

  -> El codigo NO requiere cambios
  -> Solo configurar variables de entorno
  -> El SDK de Azure es compatible con ambos
